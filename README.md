# MMCS Invest API

This is a project made by students of SFEDU MMCS. 
Provides an API for holding an info about invest portfolio.

* Ruby version 2.7.4

* Dependencies
   * gem 'faraday'
   * gem 'nokogiri'
   * gem 'jwt'
